# Manan Shah Portfolio

Personal portfolio site built with React, Vite, Tailwind CSS, and `lucide-react`.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This repo includes a GitHub Pages workflow at `.github/workflows/deploy.yml`.

If the repository is published as `Manan1121/Manan1121.github.io`, GitHub Pages will serve the site at:

- `https://manan1121.github.io/`

For a custom domain later:

1. Buy the domain from your registrar of choice.
2. In GitHub Pages settings, set the custom domain.
3. Point your DNS at GitHub Pages using the records GitHub shows in settings.
4. Add `www` forwarding or a `CNAME` record if you want both root and `www`.
